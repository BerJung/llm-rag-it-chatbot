[IT Support] VPN 접속 문제 해결 가이드

가장 자주 발생하는 VPN 이슈와 해결 방법입니다.

1. 계정/비밀번호 오류
   - 회사 계정 비밀번호가 최근에 변경되었다면, VPN 클라이언트에 저장된 비밀번호를 다시 입력해야 합니다.
   - 비밀번호가 맞는지 확인 후 다시 로그인 시도합니다.

2. 네트워크 연결 문제
   - 사내 Wi-Fi 또는 유선 네트워크에 정상적으로 연결되어 있는지 확인합니다.
   - 공용 Wi-Fi(카페, 공항 등)를 사용할 경우 일부 포트가 차단되어 VPN 연결이 안 될 수 있습니다.

3. VPN 클라이언트 오류
   - VPN 클라이언트를 완전히 종료 후 다시 실행합니다.
   - 여전히 문제가 계속되면, VPN 클라이언트를 재설치합니다.
   - 재설치 파일은 IT 포털(https://support.company.com/downloads)에서 받을 수 있습니다.

4. 권한 문제
   - 신규 입사자 또는 조직 이동 직후에는 VPN 접근 권한이 아직 반영되지 않았을 수 있습니다.
   - 이 경우 IT 티켓 시스템에서 "VPN Access Request" 티켓을 생성해 주세요.

5. 계속 실패할 때 필요한 정보
   - 사용 중인 OS (예: Windows 11, macOS 14)
   - VPN 클라이언트 이름 및 버전
   - 에러 메시지 전체 내용 (스크린샷 첨부 권장)

위 절차를 모두 진행했는데도 VPN 접속이 되지 않는다면:
- https://support.company.com 에 "VPN Issue" 티켓을 생성하고,
- 시도한 단계와 에러 메시지를 자세히 적어 주세요.
